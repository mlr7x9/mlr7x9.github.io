Canvas The first look.

One of the components I've decided to explore in this exploration is the html element canvas. The reference I used for this can be found at http://diveintohtml5.info/canvas.html. Canvas allows you to section of a portion of a webpage to use javascript inside. This is often used for graphics and games.
