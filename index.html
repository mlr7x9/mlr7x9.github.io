Grids The first component.

One of the components I've decided to explore is grids. The documentation for this can be found at http://getbootstrap.com/css/#grid. THis helps with page layout and mostly involves columns and containers to hold everything. For example, this content is in a class called col-md-7. This specifies its a column, a medium device size, and spans 8 columns. There are 12 columns across, therefore, the image next to this spans 4 columns.

Responsive images The second component.

I've also decided to explore responsive images. This will scale the image to fit the parent element. I've also included image shapes. http://getbootstrap.com/css/#images.
