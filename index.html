Grids The first component.

One of the components I've decided to explore is grids. The documentation for this can be found at http://getbootstrap.com/css/#grid. THis helps with page layout and mostly involves columns and containers to hold everything. For example, this content is in a class called col-md-7. This specifies its a column, a medium device size, and spans 8 columns. There are 12 columns across, therefore, the image next to this spans 4 columns.

Responsive images The second component.

I've also decided to explore responsive images. This will scale the image to fit the parent element. I've also included image shapes. http://getbootstrap.com/css/#images.

Canvas The first look.

One of the components I've decided to explore in this exploration is the html element canvas. The reference I used for this can be found at http://diveintohtml5.info/canvas.html. Canvas allows you to section of a portion of a webpage to use javascript inside. This is often used for graphics and games.

Click on the space above to change its background

Canvas Games A step further.

I've also decided to explore canvas games. You can use a canvas space to add javascript games. This game can be found at http://thecodeplayer.com/walkthrough/html5-game-tutorial-make-a-snake-game-using-html5-canvas-jquery. I looked at this example first. But I struggled with getting it to work.

 

SoundCloud SDK Retrieving an audio file

For this part of my exploration I've decided to look at the SoundCloud SDK and integrating SoundCloud into my site to retrieve a file. https://developers.soundcloud.com/docs/api/sdks#javascript. I explored examples like this one http://connect.soundcloud.com/examples/basic.html. I've also added a streaming button that only pulls a song, not the visuals. An interesting thing about this functionality, is that it will continue to load the song every time the button is pushed. So if the button is pushed 15 times, the song will play over itself 15 times.

SoundCloud Widget Streaming some audio via a widget

Then I took it a step further and integrated streaming of a specified playlist into my site http://connect.soundcloud.com/examples/streaming.html. The information about the widget can be found here https://developers.soundcloud.com/docs/api/sdks#javascript.
